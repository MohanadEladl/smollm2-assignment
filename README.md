# SmolLM2 Fine-Tuning Assignment

This repository contains the code and documentation for fine-tuning the SmolLM2 language model.

## Project Structure
- `*.ipynb`: Jupyter notebooks used for data preparation, training, and testing the fine-tuned model.
- `app.py`: The Gradio application source code.
- `index.html`: A static frontend for deploying to GitHub Pages, which embeds the Hugging Face Space.

## Deployment
The actual model inference is hosted for free on Hugging Face Spaces (which provides the necessary Python environment and compute resources). 

You can interact with the deployed model directly here on our GitHub Pages frontend, which acts as a bridge to the Hugging Face Space.
